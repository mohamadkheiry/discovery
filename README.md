# فرم نیازسنجی طراحی سایت آتلیه عکاسی - نسخه PHP

یک پروژه کامل PHP + MySQL برای جمع‌آوری اطلاعات نیازسنجی از مشتریان آتلیه عکاسی، همراه با داشبورد مدیریت برای مشاهده و مدیریت اطلاعات ثبت‌شده.

## ✨ امکانات

- 📝 فرم نیازسنجی کامل با ۱۳ بخش (همان طراحی اصلی HTML حفظ شده)
- 💾 ذخیره خودکار در دیتابیس MySQL
- 🔐 داشبورد مدیریت با لاگین امن (bcrypt + CSRF token)
- 🔍 جستجو و فیلتر بین رکوردها
- 👁️ مشاهده جزئیات کامل هر مشتری
- 📊 آمار کلی (کل ثبت‌شده، امروز، ۷ روز اخیر، با بودجه)
- 🗑️ حذف رکوردها با تأیید
- 📅 تاریخ شمسی در داشبورد
- 🎨 طراحی ریسپانسیو و RTL با فونت Vazirmatn

## 📋 پیش‌نیازها

- **PHP 8.0 یا بالاتر**
- **MySQL 5.7+** یا MariaDB 10.3+
- اکستنشن‌های PHP: `pdo_mysql`, `openssl`
- وب‌سرور Apache یا Nginx

## 🚀 نصب

### روش ۱: نصب خودکار (پیشنهادی)

۱. کل پوشه `studio-form-php` را در هاست یا مسیر وب‌سرور خود کپی کنید (مثلاً `htdocs/studio-form-php`).

۲. فایل `config.php` را در یک ویرایشگر باز کنید و اطلاعات دیتابیس خود را وارد کنید:

```php
define('DB_HOST', '127.0.0.1');
define('DB_PORT', '3306');
define('DB_NAME', 'studio_form');    // نام دیتابیس
define('DB_USER', 'root');           // نام کاربری دیتابیس
define('DB_PASS', '');               // رمز عبور دیتابیس
```

همچنین مقدار `CSRF_SECRET` را به یک رشته تصادفی تغییر دهید:

```php
define('CSRF_SECRET', 'یک-رشته-تصادفی-و-طولانی-اینجا');
```

۳. در مرورگر آدرس زیر را باز کنید:

```
http://localhost/studio-form-php/install.php
```

۴. در صفحه نصب:
   - دیتابیس و جداول به طور خودکار ساخته می‌شوند
   - یک فرم برای ساخت ادمین اولیه نمایش داده می‌شود
   - نام کاربری، رمز عبور و نام خود را وارد کنید

۵. پس از نصب موفق، **حتماً فایل `install.php` را حذف کنید** (برای امنیت).

۶. وارد داشبورد شوید:

```
http://localhost/studio-form-php/login.php
```

### روش ۲: نصب دستی (با SQL)

۱. فایل `schema.sql` را در phpMyAdmin یا MySQL CLI ایمپورت کنید.

۲. در صورت نیاز، یک ادمین به صورت دستی بسازید (الگوریتم bcrypt):

```sql
-- ابتدا یک فایل PHP موقت بسازید و اجرا کنید:
-- echo password_hash('your_password', PASSWORD_BCRYPT, ['cost' => 12]);
-- سپس خروجی را در جدول admins درج کنید
```

۳. فایل `config.php` را مطابق روش ۱ تنظیم کنید.

## 📂 ساختار فایل‌ها

```
studio-form-php/
├── index.php          → ریدایرکت به فرم
├── form.php           → فرم نیازسنجی (عمومی)

├── login.php          → صفحه ورود مدیر
├── logout.php         → خروج از داشبورد
├── dashboard.php      → داشبورد (لیست + جستجو + آمار)
├── view.php           → مشاهده جزئیات یک مشتری
├── delete.php         → حذف یک رکورد
├── install.php        → نصب خودکار (بعد از نصب حذف شود)
├── config.php         → تنظیمات اصلی و اتصال دیتابیس
├── schema.sql         → اسکریپت ساخت دیتابیس و جداول
├── .htaccess          → تنظیمات امنیتی Apache
└── README.md          → این فایل
```

## 🎯 نحوه استفاده

### برای مشتری (عمومی):
- به آدرس `form.php` بروید
- فرم را پر کنید و روی «ثبت اطلاعات» کلیک کنید
- پیام موفقیت نمایش داده می‌شود

### برای مدیر:
- به آدرس `login.php` بروید
- با نام کاربری و رمز عبوری که در نصب وارد کردید وارد شوید
- در داشبورد می‌توانید:
  - آمار کلی را ببینید
  - جستجو کنید (نام آتلیه، نام مدیر، تلفن، ایمیل)
  - روی 👁️ کلیک کنید تا جزئیات کامل را ببینید
  - روی 🗑️ کلیک کنید تا رکورد را حذف کنید

## 🔐 امنیت

این پروژه شامل موارد امنیتی زیر است:

- **bcrypt** برای هش کردن رمز عبور (هیچ وقت به صورت متن ذخیره نمی‌شود)
- **CSRF Token** برای جلوگیری از حملات Cross-Site Request Forgery
- **PDO Prepared Statements** برای جلوگیری از SQL Injection
- **`htmlspecialchars`** برای جلوگیری از XSS
- **Session regeneration** بعد از ورود موفق
- جلوگیری از دسترسی مستقیم به فایل‌های حساس (از طریق `.htaccess`)

### توصیه‌های امنیتی:
1. ✅ حتماً فایل `install.php` را بعد از نصب حذف کنید
2. ✅ مقدار `CSRF_SECRET` در `config.php` را به یک رشته تصادفی تغییر دهید
3. ✅ در محیط production، `display_errors` را در `config.php` روی `0` قرار دهید
4. ✅ رمز عبور قوی برای ادمین انتخاب کنید (حداقل ۱۲ کاراکتر شامل حروف، عدد و نماد)
5. ✅ از HTTPS استفاده کنید
6. ✅ به طور منظم از دیتابیس بکاپ بگیرید

## 🗃️ ساختار دیتابیس

### جدول `admins`
| فیلد | نوع | توضیح |
|------|------|-------|
| id | INT | کلید اصلی |
| username | VARCHAR(50) | نام کاربری (یکتا) |
| password_hash | VARCHAR(255) | هش bcrypt رمز عبور |
| name | VARCHAR(100) | نام مدیر |
| created_at | DATETIME | تاریخ ساخت |
| last_login | DATETIME | آخرین ورود |

### جدول `submissions`
شامل تمام فیلدهای فرم نیازسنجی (۴۰+ ستون). برای مشاهده ساختار کامل به فایل `schema.sql` مراجعه کنید.

ستون‌های مهم:
- `id` - کلید اصلی
- `studio_name`, `owner_name`, `phone`, `email` - اطلاعات تماس
- `goals`, `services`, `pages`, `features`, `trust_elements` - فیلدهای چندمقداره (CSV)
- `ip_address`, `user_agent` - متا امنیتی
- `created_at` - تاریخ ثبت

## 🐛 عیب‌یابی

### خطای «خطا در اتصال به دیتابیس»
- بررسی کنید اطلاعات `config.php` درست باشد (نام کاربری، رمز، نام دیتابیس)
- بررسی کنید MySQL در حال اجرا باشد
- بررسی کنید کاربر دسترسی به دیتابیس `studio_form` داشته باشد

### فرم ارسال می‌شود ولی ذخیره نمی‌شود
- فایل `config.php` را باز کنید و `display_errors` را روی `1` بگذارید
- صفحه را رفرش کنید و خطای نمایش‌داده‌شده را بررسی کنید

### لاگین کار نمی‌کند
- مطمئن شوید `install.php` را اجرا کرده‌اید و ادمین ساخته شده
- در صورت فراموشی رمز، یک ادمین جدید با `install.php` بسازید (ابتدا ادمین قبلی را از phpMyAdmin حذف کنید)

### صفحه سفید می‌بینم
- `error_reporting` را در `config.php` فعال کنید
- لاگ خطای PHP را بررسی کنید
